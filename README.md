📚 Cursos Gratuitos de TI - SENAI

Este projeto é um site simples desenvolvido com o objetivo de centralizar cursos gratuitos do SENAI na área de Tecnologia da Informação.

A ideia surgiu para ajudar colegas de turma a encontrarem oportunidades de capacitação de forma mais organizada, além de servir como projeto de portfólio.

🚀 Objetivo
Reunir cursos gratuitos de TI em um único lugar
Facilitar a busca por área de interesse
Compartilhar oportunidades com estudantes
Praticar desenvolvimento web com foco profissional
